# Iniciar el proyecto
Clonar el repositorio:
```bash
git clone https://github.com/max-florian/todothing.git
```
  
Dirigirse al directorio del proyecto:
```bash
cd todothing
```
  
Correr los siguientes comandos en el directorio del proyecto:
```bash
npm install
npm start
```
